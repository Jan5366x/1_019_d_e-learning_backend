Get Started
===========

Download
--------
To get started, please clone our `GitLab <https://gitlab.com/1_019_d_e_learning/backend/backend>`_.

Define config params
--------------------
Please edit config.ts to contain your MongoDB configuration

Test server
-----------
To test the server, execute
.. code::bash
   npm test

Run build
---------
To start a build, execute
.. code::bash
   npm run tsc

Start builded code
------------------
To start the builded code, run
.. code::bash
   npm run start-builded
